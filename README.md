# Generador de imagen panorámica

El generador de imagen panorámica es una herramienta escrita en python que te permite crear una imagen panorámica a partir de un video mp4.

## Required

- Python
- Numpy
- OpenCV
- Imutils

## Usage

python ./Panoramica.py --video input_video --time tiempo_captura --output output_photo

##### Required arguments:
**input_video**
Ruta del video a procesar.

**time**
Tiempo de espera para tomar un fotograma (en segundos).

**output**
Nombre y formato de imagen de la imagen de salida (en .png).

**Nota:** Debes de tener una carpeta donde se ejecute el archivo python llamada Frames.
